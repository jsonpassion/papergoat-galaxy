# PaperGoat Galaxy

AI 엔지니어 필독 논문 Top 150 인터랙티브 3D 시각화.

**Live**: https://jsonpassion.github.io/papergoat-galaxy/
**Newsletter**: https://papergoat.beehiiv.com/

![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)

## 개요

Transformer부터 TurboQuant 삼부작(2024–2026)까지, AI 엔지니어라면 반드시 알아야 할 논문
151편을 5개 평가축(Paradigm Shift, Adoption, Citation, Novelty, Timeliness)으로 점수화하여
3D 은하 형태로 시각화합니다. 논문 간 계보(lineage)는 엣지로 연결됩니다.

## 기능

- **8개 클러스터**: LLM / Architecture / Vision / Generation / Multimodal / Alignment / Agent / Efficient
- **필터**: 카테고리 · 연도 · 점수
- **lineage 엣지**: 기반 논문 ↔ 영향을 준 논문 계보 시각화
- **Papergoat 리뷰 연결**: 주요 논문 심층 해설 링크
- **반응형**: 데스크탑 / 태블릿 / 모바일
